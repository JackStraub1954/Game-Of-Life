Component State Detail

This is a description on component availability under different application states. For example, after creating a new pattern, the menu Save-As button must be available, but the Save button must not. The Rewind button must be available except when the Animation toggle button is set. A tentative description of toolbar button states is included even though the toolbar has not yet been designed.

Generator Panel

  1. Next Generation Button
    1. Not available when animation is in progress
    2. Available whenever animation is not in progress
  2. Rewind Button
    Available except when pattern is in base state. Specifically:
    1. Not available immediately after open
    2. Not available immediately after new
    3. Not available immediately after rewind
    4. Available when animation in progress
    5. No state change after save/save-as
    6. No state change after operator cell selection
  3. Generation per Second Slider:
    Always available.
  4. Animate Toggle Button:
    Always available.
